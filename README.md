# ITS Work Place Simulation API - JSON Access Guide

This project demonstrates how to retrieve data from an API that returns a `.json` response.

## Prerequisites

• Internet connection
• One of the following:
  - Browser
  - Python
  - `curl` (command line tool)
  - JavaScript (Node.js or browser)

## Base URL
  https://genausproduct.github.io/WPS-API

## Tickets Endpoint
  /tickets.json

## Active Directory Endpoint
  /active_directory.json
